# 🎱 双色球预测工具

一个基于历史数据和加权算法的双色球号码预测桌面应用，纯娱乐性质。

## ✨ 功能特性

### 📊 数据管理
- **本地数据存储**：自动保存最近 500 期历史数据到本地 CSV 文件（`~/.ssq-predictor/ssq_history.csv`）
- **智能更新机制**：
  1. 启动时优先加载本地 CSV 数据
  2. 检查最新数据的日期是否为当天
  3. 如果数据过期，自动从 datachart.500.com 获取最新数据
  4. 新数据与本地数据合并去重后保存到 CSV
- **数据持久化**：所有数据保存在用户目录下的 `.ssq-predictor/ssq_history.csv` 文件中

### 🎯 预测算法

支持两种预测策略：

1. **🔥 热号恒热**
   - 基于频率的加权算法
   - 出现频率越高的号码权重越大
   - 适合追热号的策略

2. **❄️ 冷号反弹**
   - 反向频率加权算法
   - 出现频率越低的号码权重越大
   - 适合冷号回补的策略

### 📈 分析展示

- **红球频率分析**：展示 33 个红球的历史出现频率和权重
- **蓝球频率分析**：展示 16 个蓝球的历史出现频率和权重
- **智能推荐**：通过 10,000 次迭代计算，展示概率最高的 10 组推荐号码

### 🎨 界面特点

- 现代化的渐变紫色主题
- 响应式布局，支持不同屏幕尺寸
- 直观的号码展示（红球、蓝球分色显示）
- 交互式动画效果

## 📝 使用说明

1. **首次启动**：应用会自动从网络获取最近 500 期历史数据
2. **选择算法**：点击"热号恒热"或"冷号反弹"切换预测策略
3. **查看分析**：查看红球和蓝球的频率分析数据
4. **生成预测**：点击"生成预测"按钮，获取 10 组推荐号码
5. **刷新数据**：点击"刷新数据"按钮手动更新历史数据

## 🔍 算法说明

### 频率统计
统计每个号码在历史数据中的出现次数，计算出现频率。

### 权重计算

**热号恒热算法**：
```
weight = (frequency / total)² × 100
```

**冷号反弹算法**：
```
weight = (1 - frequency / total)² × 100
```

### 号码生成
1. 基于权重进行加权随机选择
2. 执行 10,000 次迭代
3. 对每组号码计算得分
4. 选取得分最高且不重复的前 10 组
